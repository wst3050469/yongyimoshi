## v4.0.0 (全局导航 + 搜索 + 通知徽章 + 数据导出) ✅ 已发布

### 🆕 新增功能
- **🗺️ 全局导航侧边栏系统**
  - 汉堡菜单按钮，左侧滑入式侧边栏
  - 四类分组：施工管理/资源管理/质量管理/系统

- **🔍 全局搜索UI修复**
  - Header右侧搜索输入框，输入即搜（300ms防抖）
  - 复用已有 `/api/search` API + `doSearch()` 函数

- **🔔 通知未读徽章**
  - Header通知铃铛图标 + 红色数字徽章
  - 页面加载时获取未读数，每60秒自动轮询
  - 无未读时自动隐藏

- **📥 数据导出增强（CSV）**
  - 新增工人花名册、供应商清单、设备清单导出
  - 在工人/供应商/设备页面添加"导出CSV"按钮

## v3.8.0 (响应式 + 测试增强 + 工序状态机) ✅ 已发布

### 🆕 新增功能
- **📱 移动端响应式适配优化**
  - CSS全局重构：3个断点(768px/576px/400px)
  - 表格横向滚动容器(`.table-responsive`)，最小触控尺寸44px
  - 响应式工具类：`.d-none`/`.d-sm-block`/`.grid-2/3/4`/`.form-row`
  - 手机端表单垂直排列，按钮全宽，导航水平滚动

- **🧪 测试覆盖大幅增强（56→83个）**
  - 供应商管理(4)、设备管理(4)、安全检查(3)、文档(1)
  - 材料申购(2)、分包商(2)、预算(2)、通知(2)
  - 全局搜索(1)、系统统计(1)、工序状态机(5)
  - 测试隔离：每个测试使用独立临时数据库

- **🔄 工序状态机（项目阶段流转控制）**
  - 9道标准工序：基层处理→抗裂砂浆施工→...→最终验收
  - 前置校验：未完成前一道不能开始下一道
  - 支持回退：completed→in_progress
  - 自动更新项目状态（全部完成→"已完成"）
