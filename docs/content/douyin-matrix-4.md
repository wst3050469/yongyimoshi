## 四、批量生产SOP

### 每周一：定选题（30分钟）
```
从网站提取7个选题:
  周一 → construction_process.html 挑1道工序
  周二 → FAQ挑2个问题
  周三 → cases.html 挑1个案例
  周四 → knowledge-base挑1个知识点
  周五 → guide-*.html挑1个话题
  周六 → app.py挑1个功能
  周日 → 工地实拍（手机拍）
```

### 每周批量拍摄（半天搞定）
```
地点：一个工地拍全部
脚本：提前打印好
一次拍完7条口播（每条2-3遍）
一条实拍素材（多角度拍10分钟）
```

### 视频剪辑参数
```
封面:  大字标题 + 统一模板（号A蓝底、号B工地背景、号C白底、号D灰底）
时长:  15-60秒
BGM:  号A正式、号B轻快、号C舒缓、号D科技感
字幕:  必加，关键数据用黄色高亮
```
