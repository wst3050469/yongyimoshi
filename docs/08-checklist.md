# 全流程检查清单

## 一、施工前准备

### □ 材料验收
- [ ] 硅酸盐水泥42.5 - 检测报告齐全
- [ ] 中号黄沙（0.3-1.2mm） - 含泥量合格
- [ ] 细黄沙（40~70目） - 含泥量合格
- [ ] 钢纤维（剪切型） - 规格符合要求
- [ ] 永颐抗裂砂浆 YYKLSJ-05 - 检测报告齐全
- [ ] 苯丙乳液（固含量50±1%） - 检测报告齐全
- [ ] 界面剂 - 检测报告齐全
- [ ] 无机磨石专用骨料 - 粒径符合要求
- [ ] 水性锂基固化剂 - 检测报告齐全
- [ ] 纳米二氧化硅罩面剂 - 检测报告齐全

### □ 工具设备
- [ ] 永颐高精找平机器人 - 校准完成
- [ ] 磨光机 - 运转正常
- [ ] 搅拌机 - 清洁无残留
- [ ] 流动度仪 - 校准完成
- [ ] 金刚石磨片（60目/150目/500目/1000目/3000目）
- [ ] 高速抛光机（1500rpm）
- [ ] 镘刀、刮杠等手工工具

### □ 环境检查
- [ ] 环境温度 5~30℃
- [ ] 环境湿度 40~70%
- [ ] 基层含水率 ≤8%（抗裂砂浆层）
- [ ] 通风条件良好
- [ ] 照明充足

---

## 二、基层处理阶段

### □ Day 1-3
- [ ] 清除浮灰、油污
- [ ] ≥1mm裂缝注浆处理
- [ ] 洒水湿润基层（施工前2h）
- [ ] 无明水状态确认
- [ ] 界面剂涂刷（用量0.3kg/m²）
- [ ] 界面剂静置30min
- [ ] 弹线分格（间距≤8m×8m）
- [ ] 预留伸缩缝（宽5mm，深20mm）
- [ ] 基层含水率检测（塑料膜覆盖24h）

---

## 三、抗裂砂浆施工阶段

### □ Day 4-8 搅拌
- [ ] 干料混合：水泥+黄沙+钢纤维 干拌3min
- [ ] 钢纤维分3次加入
- [ ] 加入永颐抗裂砂浆 干拌2min
- [ ] 缓慢加水+苯丙乳液 湿拌5min
- [ ] 浆体扩展度检测 160-180mm

### □ 浇筑与整平
- [ ] 机器人摊铺找平（误差≤2mm）
- [ ] 边角刮杠刮平
- [ ] 磨光机收光（初凝前完成）
- [ ] 切缝处理（终凝后24h内）

### □ 养护
- [ ] 浇筑后2h覆盖PE膜
- [ ] 喷水保持湿润
- [ ] 养护7天
- [ ] 温度控制（冬季≥5℃，夏季防暴晒）

### □ 验收
- [ ] 7D抗压≥25MPa
- [ ] 表面无裂纹（≤0.2mm）
- [ ] 平整度≤2mm

---

## 四、面层施工阶段

### □ Day 16 界面处理
- [ ] 基层抗压≥20MPa
- [ ] 基层含水率≤6%
- [ ] 滚涂界面剂第1遍
- [ ] 间隔2h滚涂第2遍
- [ ] 界面剂用量0.4kg/m²
- [ ] 界面剂完全固化（触干≤4h）

### □ Day 17-22 面层搅拌与浇筑
- [ ] 干混料+80%B组分水 搅拌2min
- [ ] 倒入剩余20%水 搅拌3min至泛光泽
- [ ] 加入无机磨石专用骨料 搅拌1min
- [ ] 底层施工（如需要）：厚度6mm无大骨料
- [ ] 面层摊铺（机器人，误差1~2mm）
- [ ] 层间间隔≤1h
- [ ] 初凝前镘刀收光2次
- [ ] 环境湿度≤70%

### □ 养护
- [ ] 浇筑后4h覆盖无纺布+PE膜
- [ ] 喷水养护3~7天
- [ ] 恒温20±5℃

### □ 验收
- [ ] 3D抗压≥25MPa
- [ ] 表面无起砂、脱粒

---

## 五、打磨抛光阶段

### □ 粗磨（24h后）
- [ ] 60目金刚石磨片
- [ ] 去除浮浆
- [ ] 暴露骨料30%

### □ 中磨（7天后）
- [ ] 150目磨片
- [ ] 500目磨片
- [ ] 消除划痕

### □ 精磨（28天后）
- [ ] 1000目磨片
- [ ] 3000目磨片
- [ ] 粗糙度Ra≤0.2μm
- [ ] 打磨深度≤3mm
- [ ] 边角手工精磨处理

---

## 六、密封固化阶段

### □ 固化剂
- [ ] 滚涂第1遍（用量0.25kg/m²）
- [ ] 间隔4h滚涂第2遍
- [ ] 固化48h
- [ ] 高速抛光（1500rpm + 白色垫片）

### □ 罩面（可选）
- [ ] 喷涂纳米二氧化硅（用量0.1kg/m²）
- [ ] 光泽度≥70GU

### □ 最终验收
- [ ] 抗污性：咖啡/酱油24h不渗透
- [ ] 光泽度≥40GU（未罩面）/ ≥70GU（罩面）
- [ ] 28D抗压≥50MPa
- [ ] 整体无裂缝
