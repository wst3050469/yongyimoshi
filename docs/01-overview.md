# 项目概述与设计指标

## 1. 结构设计

| 层次 | 材料 | 厚度 |
|------|------|------|
| 面层 | 无机金磨石 | 15mm |
| 基层 | 抗裂砂浆 | 30~50mm |

## 2. 性能指标

| 指标 | 要求 | 检测标准 |
|-----|------|---------|
| 抗压强度 | ≥60 MPa | GB/T 50081 |
| 抗折强度 | ≥8 MPa | GB/T 50081 |
| 耐磨性 | ≤0.05 g/cm² | GB/T 12988 |
| 防滑等级 | R10 | DIN 51130 |
| 光泽度 | ≥70 GU（罩面） | GB/T 13891 |
| 裂缝 | 无 | 目测 |

## 3. 工期总表

| 阶段 | 天数 | 说明 |
|------|------|------|
| 基层处理 | 3天 | 清理、修补、界面处理 |
| 抗裂砂浆施工 | 5天 | 搅拌、浇筑、整平、切缝 |
| 养护 | 7天 | 覆膜喷水养护 |
| 无机磨石施工 | 7天 | 界面处理→浇筑→打磨→固化 |
| **总计** | **约22天** | |

## 4. 环境要求

| 参数 | 要求 |
|------|------|
| 施工温度 | 5~30℃ |
| 环境湿度 | 40~70% |
| 基层含水率 | ≤8%（抗裂砂浆层）/ ≤6%（面层） |
