# 风险应急预案

## 1. 常见风险与应对

| 风险问题 | 原因分析 | 应急预案 |
|---------|---------|---------|
| **颜色不均** | 颜料分散不均匀 | 立即停止施工，检查颜料预混工艺，调整搅拌时间 |
| **骨料沉降** | 浆体触变性不足 | 添加0.1%膨润土触变剂，缩短浆体开放时间至1.5h |
| **低温施工** | 温度低于5℃ | 添加早强剂（甲酸钙0.5%），苯丙乳液替换为低温型（Tg≤0℃） |
| **表面泌水** | 环境湿度过高 | 控制环境湿度≤70% |
| **冷接缝** | 层间间隔超过1h | 严格控制层间间隔≤1h |
| **骨料脱落** | 打磨深度超过3mm | 控制打磨深度≤3mm |
| **表面气孔** | 收光不及时 | 初凝前镘刀收光2次 |

## 2. 环境异常预案

### 2.1 高温施工（>30℃）

| 措施 | 说明 |
|------|------|
| 遮阳防暴晒 | 搭设遮阳棚 |
| 调整凝结时间 | 使用缓凝剂调节 |
| 加强保湿 | 增加喷水频率 |

### 2.2 低温施工（<5℃）

| 措施 | 说明 |
|------|------|
| 添加早强剂 | 甲酸钙0.5% |
| 乳液替换 | 苯丙乳液替换为低温型（Tg≤0℃） |
| 保温措施 | 覆盖保温材料，确保≥5℃ |

### 2.3 高湿环境（>70%）

| 措施 | 说明 |
|------|------|
| 面层施工限制 | 环境湿度≤70% |
| 除湿措施 | 使用除湿设备 |
| 延长养护 | 适当延长养护时间 |

## 3. 质量缺陷应急处理

### 3.1 裂缝处理

| 裂缝宽度 | 处理方法 |
|---------|---------|
| <0.2mm | 可接受，无需处理 |
| ≥0.2mm | 注浆修补 |

### 3.2 表面缺陷

| 缺陷类型 | 处理方法 |
|---------|---------|
| 起砂 | 重新涂刷固化剂 |
| 脱粒 | 局部修补后重新打磨 |
| 光泽度不足 | 重新抛光或增加罩面层 |
